Chrome Extension

Tuya Desk

Compact Chrome extension to control Tuya light switches by channel, with synced setup, fast popup refresh and a clean user mode.

Privacy Policy GitHub Repository

What it does

  • Connects to Tuya Cloud using the region configured by the user.
  • Detects 1, 2, 3 and 4 gang switches automatically.
  • Controls each device channel independently.
  • Syncs setup and display order through Chrome sync storage.
